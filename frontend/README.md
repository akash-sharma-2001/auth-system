# Auth System (Full Stack MERN)

A full-stack authentication system built using the MERN stack.
This project includes secure user authentication with JWT stored in HTTP-only cookies, protected routes, forgot password functionality and a modern frontend UI.

---

## Features

### Authentication
- User Signup & Login
- Password hashing using bcryptgit
- JWT based authentication
- Token stored in HTTP-only cookies
- Logout functionality
- Protected routes (frontend & backend)
- Forgot password using email (NodeMailer)

### Email
- Email sending using Nodemailer
- Used for notifications / verification

### Frontend
- Responsive UI using Tailwind CSS
- API handling with Axios
- Notifications using React Toastify
- Protected routes using React Router
